## 1. ЧТО ЭТО ЗА ПРОЕКТ

Это ИИ-бот для продакшн-студии **Tot Media** (Казахстан), построенный на платформе OpenClaw. Бот работает одновременно на **двух каналах** с **разными ролями**:

| Канал        | Роль ИИ                                    | Цель                                                                                                                                                                  |
| ------------ | ------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **WhatsApp** | **Алимжан** — менеджер по B2B/B2C продажам | Консультировать клиентов по прайсу, собирать брифы, закрывать сделки и передавать лиды в Telegram-группу администраторов                                              |
| **Telegram** | **Личный Секретарь / ИИ-Аналитик**         | Выполнять поручения владельца студии: парсинг Instagram конкурентов, расчет маржинальности проектов, генерация коммерческих предложений, поиск информации в интернете |

## 3. ВСЕ 6 ИНСТРУМЕНТОВ (tools.ts)

Файл: `extensions/totmedia-sales/src/tools.ts`

| #   | Инструмент                  | Канал    | Что делает                                                                                                                                                                         |
| --- | --------------------------- | -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1   | `forward_lead_to_admin`     | WhatsApp | Собирает полный бриф (имя, задача, цель, бюджет, телефон) и отправляет его в Telegram-группу администраторов (chat_id: `-5155469349`)                                              |
| 2   | `escalate_to_human`         | WhatsApp | Если клиент злится или просит оператора — отправляет алерт лично владельцу (chat_id: `5459177374`)                                                                                 |
| 3   | `analyze_instagram_profile` | Telegram | Парсит Instagram через Apify API (актор `apify/instagram-scraper`). Извлекает био, подписчиков, лайки, комментарии и текст последних 5 постов. ИИ потом делает аналитический отчет |
| 4   | `calculate_project_margin`  | Telegram | Калькулятор рентабельности: принимает выручку и расходы (команда, студия, транспорт), считает чистую прибыль и % маржи. Предупреждает если маржа < 30%                             |
| 5   | `search_google`             | Telegram | Ищет в Google (библиотека `googlethis`), возвращает топ-5 результатов с заголовками, URL и описаниями                                                                              |
| 6   | `read_webpage`              | Telegram | Скачивает HTML страницы, очищает от мусора через `cheerio`, возвращает чистый текст (лимит 8000 символов)                                                                          |

## 4. СИСТЕМНЫЙ ПРОМПТ (SOUL.md)

Файл: `~/.openclaw/workspace/SOUL.md`

### Структура:

1. **Мульти-канальная маршрутизация** — ИИ определяет платформу (WhatsApp/Telegram) и переключает роль
2. **Роль Алимжана (WhatsApp):**
   - Tone of Voice: вежливый эксперт, без давления
   - Воронка продаж: Приветствие → Квалификация (по одному вопросу!) → Оффер → Стоимость → CTA → Follow-up
   - Guardrails (жесткие правила):
     - **Пинг-понг**: максимум 1 вопрос за сообщение
     - **Списки запрещены** при квалификации (только при презентации тарифов)
     - **Анти-галлюцинация**: нельзя придумывать цены, которых нет в прайсе
     - **Scope**: только фото/видеопродакшен
   - База знаний (Прайс-лист в тенге)
   - Главная цель: собрать бриф и вызвать `forward_lead_to_admin`
3. **Роль Секретаря (Telegram):**
   - Инструкции для анализа Instagram, генерации КП, расчета смет
   - Протокол интернет-серфинга: search_google → read_webpage → аналитический ответ

- **WhatsApp**: включен, dmPolicy=open, allowFrom=["*"]
- **Telegram**: включен, dmPolicy=open, streaming=partial
- **Плагин**: загружается из `c:/Users/kural/Desktop/Client_Bot/extensions/totmedia-sales`
- **Gateway**: mode=local, auth=token

## 8. КАК ЗАПУСТИТЬ ПРОЕКТ

### Первый запуск (на новом компьютере):

```bash
# 1. Клонировать репозиторий
git clone https://github.com/ruslan20200/TotMedia-Bot.git Client_Bot
cd Client_Bot

# 2. Установить зависимости движка
pnpm install

# 3. Установить зависимости плагина
cd extensions/totmedia-sales
pnpm install
cd ../..

# 4. Скопировать папку .openclaw от рабочего компьютера в ~/.openclaw/
# (содержит openclaw.json, SOUL.md, сессию WhatsApp)

# 5. Запустить шлюз
pnpm openclaw gateway --port 18789
```

### Повседневный запуск:

```bash
pnpm openclaw gateway --port 18789
```

### После изменений в коде:

```bash
# Остановить шлюз (Ctrl+C), затем перезапустить:
pnpm openclaw gateway --port 18789
```

### Сброс памяти ИИ (если бот "запутался"):

```powershell
Remove-Item -Recurse -Force C:\Users\kural\.openclaw\agents\main\sessions\*
```
